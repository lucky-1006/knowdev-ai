Please do **NOT** report security vulnerabilities in public GitHub issues.

> [!IMPORTANT]
> The full security disclosure policy, PGP keys, supported versions list, and deployment best practices are available in our [Security Guide](file:///c:/Users/STAR/Documents/CodeEngine/docs/SECURITY.md).

## Quick Reporting Guide

If you discover a vulnerability:
* **Contact**: Send email details directly to [security@knowdev.ai](mailto:security@knowdev.ai).
* **Details**: Describe the vulnerability, PGP-encrypting the report if possible, and include details on how to reproduce the issue.
